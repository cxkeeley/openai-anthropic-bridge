**Project**: OpenAI-Anthropic Bridge for JiuTian (九天) Model

**Purpose**: Create a bridge application that allows the JiuTian AI model to be used with both the Anthropic and OpenAI API formats. The bridge translates between these API formats, enabling seamless integration with various AI development tools and platforms.

**Key Features**:
- Anthropic API compatible endpoint at `/v1/messages`
- OpenAI API compatible endpoint at `/v1/chat/completions`
- Rate limiting (disabled by default)
- Model parameters configuration
- Streaming support
- Tool call handling
- System override support

**Current Status**: Production-ready, all 69 tests passing

**Branch**: prod (deployed), main (stable)
